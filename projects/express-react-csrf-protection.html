Express-React CSRF Protection

Express.js • React.js • Sécurité CSRF

Ce projet démontre l’intégration entre Express.js (backend) et React.js (frontend) avec une protection CSRF complète pour sécuriser les requêtes entre le client et le serveur. Il illustre comment générer, valider et renouveler les tokens CSRF afin d’empêcher toute attaque de type Cross-Site Request Forgery.

Objectifs du projet

  • Protéger les requêtes entre React et Express contre les attaques CSRF
  • Implémenter la génération automatique de tokens sécurisés
  • Illustrer les bonnes pratiques de sécurité côté client et serveur

Fonctionnalités clés

  • Protection CSRF intégrée sur toutes les requêtes POST
  • Gestion dynamique du token CSRF côté frontend
  • Middleware Express pour validation automatique des tokens
  • Configuration simple pour les projets MERN

Détails techniques & bonnes pratiques

  • Utilisation de csurf pour la génération et la vérification des tokens
  • Intégration avec fetch et les en-têtes personnalisés côté React
  • Serveur Express configuré avec helmet et cors
  • Déploiement possible sur Vercel ou Render
