Recettes Cuisine — Application mobile

Flutter • API Spoonacular • Recherche par ingrédients

Application mobile développée en Flutter permettant de rechercher des recettes selon les ingrédients disponibles. Intégration de l'API Spoonacular pour récupérer images, ingrédients et instructions. L'application propose aussi une gestion des favoris et une navigation fluide pour une bonne expérience utilisateur mobile.

Flutter Spoonacular API Dart UX Mobile

Objectifs du projet

  • Permettre la recherche de recettes par ingrédient disponible
  • Fournir des instructions claires et illustrées pour chaque recette
  • Offrir une expérience mobile fluide avec gestion des favoris

Fonctionnalités clés

  • Recherche intelligente par ingrédients (matching et suggestions)
  • Affichage des étapes de préparation, temps et ingrédients nécessaires
  • Favoris sauvegardés localement pour accès hors‑ligne
  • Pagination / lazy loading des résultats pour performance

Détails techniques & bonnes pratiques

  • Gestion des requêtes à l'API Spoonacular avec cache et gestion d'erreurs
  • Architecture modulaire Flutter (Widgets réutilisables, services)
  • Tests via flutter_test pour les composants critiques
  • Optimisation des images et lazy loading pour réduire la consommation mobile
